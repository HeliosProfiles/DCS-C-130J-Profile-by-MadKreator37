# Helios Profile for _aircraft by _author
This is a profile for Helios Virtual Cockpit to support the _aircraft.

### Creating a New Profile

#### Login to GitHub

When logged in, navigate to the actions section of the Templates repository of the **HeliosProfiles** organisation [Action To Create New Profile Repository](https://github.com/HeliosProfiles/Template/actions/workflows/Create%20New%20Profile%20Repo.yml)

<img width="1036" height="515" alt="image" src="https://github.com/user-attachments/assets/e490d889-b0a3-4100-9236-7358db601f55" />

Click the wee triangle beside the **Run workflow** button on the right had side of the screen.

An option panel should appear

<img width="296" height="473" alt="image" src="https://github.com/user-attachments/assets/7d2db8db-3b44-4fc5-82cd-e2cfc52952d9" />

Once the required information has been entered, press the green **Run workflow** button at the bottom of the drop down.

About 30 seconds or so later, you should have a new repository set up, ready for you to clone onto your machine with Github Desktop, and populate with
your new profile.

You can locate your new repository by going to the [repositories section](https://github.com/orgs/HeliosProfiles/repositories) of the HeliosProfiles organisation.  To clone your new repository, you should got to the **Code** green button on the main screen for your new repository, and copy the URL that appears there into your Github Desktop.
